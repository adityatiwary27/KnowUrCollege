# KnowUrCollege

KnowUrCollege is a full-stack web application designed for college communities, enabling users to share posts, interact through likes and comments, and stay connected with campus activities.

## 🚀 Features
- User Registration & Login
- Create and View Posts
- Like and Comment on Posts
- REST APIs with Spring Boot

## 🛠 Tech Stack
- Backend: Spring Boot (Java)
- Frontend: React (to be added)
- Database: MySQL

## ⚙️ Setup Instructions
1. Clone the repository
2. Configure MySQL in application.properties
3. Run:
   mvn spring-boot:run
   
Alternatively on Windows you can use the helper script which will stop any process using port 8080 and start the app:

```powershell
scripts\start.ps1
```
